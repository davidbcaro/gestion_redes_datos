¿Qué es una dirección IPv4?
Una dirección IPv4 es un número único de 32 bits que identifica cada dispositivo en una red, como si fuera su número de teléfono. Se escribe como 4 grupos de números del 0 al 255, separados por puntos. Ejemplo: 192.168.1.1
🏠
Analogía — Dirección Postal
Imagina que la dirección IP es como la dirección de tu casa: País · Ciudad · Calle · Número. De la misma forma, una IP tiene una parte que identifica la RED (como el país y la ciudad) y otra que identifica al HOST (como la calle y el número de casa). El prefijo (ej. /24) dice cuántos bits son de "red" y cuántos de "host".

¿Qué es una dirección IPv6?
IPv4 se está quedando sin direcciones — solo hay ~4,300 millones y ya se agotaron. IPv6 tiene 340 sextillones de direcciones (un 3 seguido de 38 ceros). En lugar de 32 bits, usa 128 bits escritos en 8 grupos hexadecimales separados por dos puntos. Ejemplo: 2001:0db8:85a3::8a2e:0370:7334
🏙️
Analogía — De casa a ciudad galáctica
IPv4 es como tener códigos postales de 4 dígitos — pronto se acaban. IPv6 es como si cada átomo del universo tuviera su propio número de teléfono y aún sobraran números. La dirección tiene dos partes: Prefijo de Red (los primeros 64 bits, como el país+ciudad) e Interface ID (los últimos 64 bits, como el número de casa).

Tipos de dirección IPv6 — ¿Cuál necesitas?
🌍
Global Unicast
2000::/3
Equivale a una IP pública de IPv4. Enrutable en internet.
🔗
Link-Local
FE80::/10
Solo funciona en el enlace local (tu red directa). Como APIPA en IPv4.
🏢
Unique Local
FC00::/7
Equivale a las IPs privadas de IPv4. Usada dentro de organizaciones.
🔄
Loopback
::1/128
El mismo dispositivo. Equivale a 127.0.0.1 en IPv4.
📡
Multicast
FF00::/8
Para grupos de dispositivos. Reemplaza al broadcast de IPv4.
✏️
Personalizado
Manual

ℹ️
Las Global Unicast son las más comunes — funcionan en internet igual que las IPs públicas de IPv4. Tu ISP te asigna un prefijo /48 o /56 y tú subneteás desde ahí.

📐 Las 2 Reglas de Compresión de IPv6 — ¡Muy importante!
1️⃣
Omitir ceros a la izquierda dentro de cada grupo.
0db8 → db8  ·  0001 → 1  ·  0000 → 0
2️⃣
Reemplazar grupos consecutivos de 0000 con :: (solo una vez).
2001:0db8:0000:0000:0000:0000:0000:0001
→ 2001:db8::1
Ejemplos reales de compresión:

→

→
::1

→

¿Cómo funciona el subnetting?
1
Tienes una red grande: 192.168.1.0/24 con 256 direcciones disponibles.
2
Para crear 4 subredes, necesitas 2 bits adicionales de la parte de host (2² = 4 subredes).
3
El nuevo prefijo es /24 + 2 = /26. Cada subred tendrá 64 IPs y 62 hosts útiles.
4
Las subredes son contiguas: .0, .64, .128, .192 — separadas por 64 posiciones.
